##  Miniguia de Estudo – Full Stack JavaScript

##  1. Resumo

**Full Stack JavaScript** é o desenvolvimento de aplicações usando JavaScript tanto no front-end quanto no back-end.

###  Partes principais:

- **Front-end:** o que o usuário vê (interface)  
- **Back-end:** onde ficam as regras e o processamento  
- **Banco de dados:** onde as informações são guardadas  
- **API:** conexão entre front e back  

###  Como funciona:

1. Usuário interage com a interface  
2. Front-end envia pedido ao back-end  
3. Back-end processa e busca dados  
4. Resposta volta para o front-end  
5. Tela é atualizada  

---

##  2. Glossário

- **Front-end:** parte visual  
- **Back-end:** parte lógica  
- **API:** ligação entre sistemas  
- **Node.js:** JavaScript no servidor  
- **Banco de dados:** armazenamento de dados  
- **JSON:** formato de dados  
- **CRUD:** criar, ler, atualizar, deletar
